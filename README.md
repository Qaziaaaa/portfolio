## Getting Started

```bash
npm install
cp .env.example .env   # Add your API keys
npm run dev
```

**Required environment variables:**
```env
VITE_JINA_API_KEY=your_jina_api_key_here   # Free at jina.ai
VITE_GROQ_API_KEY=your_groq_api_key_here   # Free at console.groq.com
```
